# vimrc-for-python
my vimrc for python

## Set up Vundle
` git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim`

## Download `vimrc` to your home as `.vimrc`

## Install Plugins:

Launch `vim` and run `:PluginInstall`

To install from command line: `vim +PluginInstall +qall`
